# DataLens — AI Data Dashboard

Upload your data and let AI explore, analyze, and visualize it for you.

## Features

- **Drag & Drop Upload** — CSV and Excel files, or paste raw text data
- **AI-Powered Analysis** — Agent explores data with built-in query tools, finds patterns and insights
- **Interactive Charts** — Plotly visualizations generated by AI
- **Chat with Data** — Ask questions about your dataset in natural language
- **Dark Mode** — Toggle between light and dark themes
- **Modern UI** — Glassmorphism, smooth animations, responsive design

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **UI:** Tailwind CSS v4 + shadcn/ui + Framer Motion
- **Charts:** Plotly.js (react-plotly.js)
- **AI:** OpenRouter API — 13 models selectable from the UI (GPT-4, Claude, Gemini, Llama, etc.)
- **Data Parsing:** PapaParse (CSV) + SheetJS (Excel)
- **Deployment:** Vercel

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Configure environment

Create `.env.local`:

```
OPENROUTER_API_KEY=sk-or-v1-your-key-here
```

Get your key at [openrouter.ai](https://openrouter.ai/keys)

### 3. Run development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### 4. Deploy to Vercel

```bash
npx vercel
```

Or connect your GitHub repo to Vercel for automatic deployments.

## How It Works

1. **Upload** — Drop a CSV or Excel file (or paste text data)
2. **Agent EDA** — AI explores your data using query tools (groupby, filter, correlation), computes statistics, finds patterns
3. **Plan** — Agent describes which charts it will create and why
4. **Render** — Agent creates Plotly charts via tool calls, charts appear as interactive visualizations
5. **Chat** — Ask follow-up questions about your data
