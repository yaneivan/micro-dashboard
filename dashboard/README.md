# DataLens — AI Data Dashboard

Upload your data and let AI explore, analyze, and visualize it for you.

## Features

- **Drag & Drop Upload** — CSV and Excel files up to 50MB
- **AI-Powered Analysis** — Agent explores data with pandas, finds patterns and insights
- **Interactive Charts** — Plotly visualizations generated by AI
- **Chat with Data** — Ask questions about your dataset in natural language
- **Dark Mode** — Toggle between light and dark themes
- **Modern UI** — Glassmorphism, smooth animations, responsive design

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **UI:** Tailwind CSS v4 + shadcn/ui + Framer Motion
- **Charts:** Plotly.js (react-plotly.js)
- **AI:** OpenRouter API (supports Claude, GPT-4, Gemini, etc.)
- **Data Parsing:** PapaParse (CSV) + SheetJS (Excel)
- **Deployment:** Vercel

## Getting Started

### 1. Install dependencies

```bash
cd dashboard
npm install
```

### 2. Configure environment

Create `.env.local`:

```
OPENROUTER_API_KEY=sk-or-v1-your-key-here
OPENROUTER_MODEL=anthropic/claude-sonnet-4
```

Get your key at [openrouter.ai](https://openrouter.ai/keys)

Available models: `anthropic/claude-sonnet-4`, `openai/gpt-4.1-mini`, `google/gemini-2.5-flash`, `meta-llama/llama-4-maverick`, and [many more](https://openrouter.ai/models).

### 3. Run development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### 4. Deploy to Vercel

```bash
npx vercel
```

Or connect your GitHub repo to Vercel for automatic deployments.

## How It Works

1. **Upload** — Drop a CSV or Excel file
2. **Agent EDA** — AI explores your data using pandas, computes statistics, finds correlations
3. **Plan** — Agent describes which charts it will create and why
4. **Render** — Agent generates Plotly code, charts appear as interactive visualizations
5. **Chat** — Ask follow-up questions about your data
